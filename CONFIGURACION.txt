ESPECIFICACIONES RECOMENDADAS DE IMÁGENES
==========================================

Logo:
- Tamaño: 50x50 px
- Formato: PNG (con transparencia recomendada)
- Color: Que contraste con fondo oscuro

Imágenes de Productos:
- Tamaño: Mínimo 400x400 px (preferiblemente 600x600+ para galería)
- Formato: JPEG o PNG
- Fondo: Preferiblemente neutro/blanco
- Resolución: Optimizada para web (72-96 DPI)
- Peso: <500 KB por imagen

Proporciones:
- Cuadrada (1:1) para tarjetas
- La galería se adapta automáticamente


CÓMO AGREGAR NUEVAS IMÁGENES
=============================

1. Coloca la imagen en la carpeta correspondiente:
   FrontEnd/Img/[Categoria]/imagen.jpg

2. Actualiza el archivo Js/data/productos.js:

   {
       id: 1,
       nombre: "Nombre del Producto",
       precio: 259.99,
       imagen: "../Img/Silenciadores/silenciador1.jpg",    ← Imagen principal (tarjeta)
       imagenes: [                                          ← Galería de imágenes
           "../Img/Silenciadores/silenciador1.jpg",
           "../Img/Silenciadores/silenciador1-2.jpg",
           "../Img/Silenciadores/silenciador1-3.jpg"
       ],
       descripcion: "Descripción...",
       especificaciones: { ... }
   }

3. Guarda los cambios y recarga el navegador.


AGREGAR NUEVAS CATEGORÍAS
=========================

1. Abre FrontEnd/Index.html
2. En la sección "categories-grid", agrega una nueva tarjeta:

   <div class="category-card" onclick="navigateToCategory('NuevaCategoria')">
       <div class="category-icon">
           <i class="fas fa-[icon]"></i>
       </div>
       <h3>Nueva Categoría</h3>
       <p>Descripción de la categoría</p>
       <span class="view-more">Ver Productos <i class="fas fa-arrow-right"></i></span>
   </div>

3. Abre FrontEnd/Js/data/productos.js
4. Agrega un nuevo array en el objeto "productos":

   NuevaCategoria: [
       {
           id: 16,
           nombre: "Producto 1",
           precio: 99.99,
           imagen: "../Img/NuevaCategoria/imagen.jpg",
           imagenes: ["../Img/NuevaCategoria/imagen.jpg"],
           descripcion: "Descripción...",
           especificaciones: { propiedad: "valor" }
       },
       // ... más productos
   ]

5. Crea la carpeta FrontEnd/Img/NuevaCategoria/ y agrega las imágenes.

SOLUCIÓN DE PROBLEMAS
=====================

Las imágenes no se cargan:
→ Verifica que la ruta sea correcta
→ Asegúrate que el nombre del archivo coincida (mayúsculas/minúsculas)
→ Usa placeholders automáticos si la imagen falla

El modal no abre:
→ Verifica que categoria.js esté cargado
→ Abre la consola del navegador (F12) para ver errores

Estilos no se aplican:
→ Limpia la caché (Ctrl+Shift+R)
→ Verifica que las rutas a CSS sean correctas
→ Revisa si hay errores de sintaxis CSS

Menú móvil no funciona:
→ Verifica que el ID "menuToggle" exista en el HTML
→ Comprueba que Index.js esté cargado
